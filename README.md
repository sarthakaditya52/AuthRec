# Django Web App for
https://github.com/UtkarshBelwal/Speaker-Recognition-and-Authentication

## Requirements
- Python version : 3.7
- pipenv

## Libraries used
- django
- numpy
- scipy
- sklearn
- python-speech-features
- sounddevice
- matplotlib

## How to run:
```bash
  # Install pipenv
  pip3 install pipenv

  # Create a virtual environment
  pipenv shell

  # Install All the libraries
  pipenv install

  # Run server
  cd AuthRec
  python manage.py runserver
```

### The webapp will be available at :
    localhost:8000
